# ArrivalRitual

Standalone HarmonyOS watch app for travel guidance, cultural alerts, and quick conversation help.

## Project

- `watch-app/` contains the full app.
- `docs/` keeps submission and reference materials relevant to the watch app.

## Quick Start

1. Open `watch-app/` in DevEco Studio 5.0+.
2. Make sure your local Huawei SDK is installed and configured in `watch-app/local.properties`.
3. Copy `watch-app/entry/build-profile.json5.example` to `watch-app/entry/build-profile.json5`.
4. Add your API keys, then run the `entry` module on a HarmonyOS watch emulator or device.

## Notes

- The repo no longer includes the old Android controller app or web prototype.
- The watch app is intended to run standalone without a companion phone app.
